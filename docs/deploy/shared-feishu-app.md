# Shared Feishu App Guide Retired

Parsar no longer documents or distributes a shared internal Feishu
application. Shared credentials, callback allowlists, and app ownership belong
in the owning organization's private operations documentation.

Use the current guides instead:

- Configure production Feishu OAuth with [`feishu-prod.md`](feishu-prod.md).
- Review workspace and Agent message routing in
  [`../feishu-routing.md`](../feishu-routing.md).
- Follow the general deployment process in
  [`deploy-runbook.md`](deploy-runbook.md).
- Configure workspace Feishu messaging connectors in the Parsar admin UI.
