# LAN deployment guide moved

This guide is retained as a migration page so existing links and bookmarks do
not break. Its previous instructions described an obsolete deployment model
and environment-based connector configuration.

Use the current documentation instead:

- [`INSTALL.md`](../../INSTALL.md) for the supported Docker Compose install and
  upgrade flow.
- [`deploy-runbook.md`](./deploy-runbook.md) for deployment configuration,
  operations, verification, and troubleshooting.

Workspace connectors such as Feishu, Slack, and Discord are configured in the
Parsar admin UI rather than through deployment-specific bot environment
variables.
